## 📖 Как пользоваться / How to use

1. **Обновите список:** Нажмите кнопку обновления, чтобы получить свежие адреса.
2. **Выберите сервер:** Ориентируйтесь на минимальный пинг (ms).
3. **Подключитесь:** Нажмите `Connect`, и приложение перенаправит вас в Telegram для активации.
4. **Важно:** Для корректного обновления списка рекомендуем временно отключить сторонние VPN.
